# CredAi
## HelloWorld.java
A simple Java program that prints "Hello, World" to the console.
Concepts covered:
Basic Java program structure
main() method
Console output using System.out.println()
## ArrayListExample.java
A Java program that demonstrates the use of ArrayList to store and display a list of strings.
Concepts covered:
ArrayList creation
Adding elements using add()
Printing ArrayList elements
Sample elements: Apple, Orange, Mango
## LinkedListExample.java
A Java program that demonstrates the use of LinkedList to store and print elements.
Concepts covered:
LinkedList creation
Adding elements
Printing LinkedList contents
Sample elements: One, Two, Three, Four, Five
## StackExample.java
A Java program that demonstrates Stack operations using the Stack class.
Concepts covered:
Stack creation
push() operation
pop() operation
LIFO (Last In, First Out) behavior
The program pops and prints elements until the stack becomes empty.
## QueueExample.java
A Java program that demonstrates the use of Queue with a PriorityQueue implementation.
Concepts covered:
Queue interface
PriorityQueue implementation
Adding elements using add()
Printing queue elements
Elements are stored based on priority
